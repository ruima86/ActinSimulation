These files are the code for the simulation of actin filaments organizing around a cylinder by crosslinking proteins. Descriptions of the files are list below.

main.cpp:                
The main program

BasicLinearAlgebra.h:
Library including some self-written routines for basic operations of vectors and matrices

filament.h:
Class file defining the object filament

mvnrnd.h:
functions to generate multi-dimensional random numbers of Guassian distribution.

node.h:
Class file defining the object Connection which is to model the cross.
