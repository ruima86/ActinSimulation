List of files

1. Connection.cpp and  Connection.hpp

Define the methods and properties of class Connection, which saves information 
about a crosslinker.

2. filament.cpp and filament.hpp

Define the methods and proeprties of class filament, which saves information about
a filament.


3. main.cpp

This is the file to run to generate simulated trajectores of actin filaments. The 
resulting data file contains the all the information of filaments, including their
center of mass, orientation, rotation, length and label.

4. CalculateStatistics.m

This is the file to Calculate the metrics used to analyze the structure of actin filaments.
